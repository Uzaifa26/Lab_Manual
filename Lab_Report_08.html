Reflection:

Working on this interactive webpage helped me understand event handling in a much more practical way. Instead of just reading about events like click, hover, and double-click, I actually used them to make different elements respond with color changes, size adjustments, movements, and shape transformations. Designing these interactions made the connection between JavaScript events and DOM manipulation very clear to me, especially when using the this keyword to refer to the active element. Through trial and error, I learned how each event can create a unique user experience and how important it is to write clean, responsive, and easy-to-understand code. Overall, this open-ended experiment improved both my creativity and my confidence in handling events in JavaScript
